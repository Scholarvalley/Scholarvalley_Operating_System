BACKGROUND IMAGES FROM ARCHIVE
==============================

Drop your background pictures here so the site matches the archive.

Suggested filenames (use whatever you save from the archive):

  hero-bg.jpg     - Hero / main banner background
  body-bg.png     - Page background (tiled or full)
  about-bg.jpg    - About section background
  header-bg.png   - Optional header background

After adding files, reference them in CSS or HTML, e.g.:
  url('/static/assets/hero-bg.jpg')

See ARCHIVE_COPY_GUIDE.md in the project root for how to copy every page
and images from the Wayback Machine.
